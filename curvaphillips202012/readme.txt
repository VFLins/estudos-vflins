Este estudo foi feito com dados retirados do SIDRA

Foram utilizados dados trimestrais de desemprego para todas as idades legais, quanto ao IPCA, foi utilizada a inflação acumulada até o último mês de cada trimestre
O período de tempo retirado foi 2012 (1º trimestre) até 2020 (3º trimestre)

Pode haver alguma diferença no resultado do script, dependendo da versão dos pacotes utilizados.
